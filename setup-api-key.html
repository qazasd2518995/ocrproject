🔑 Groq API Key 設定

設定後即可使用 AI 智能發票格式化功能

📋 1. 取得免費 Groq API Key

  1. 前往 Groq 官方網站註冊
  2. 使用 Google 或 GitHub 帳號登入
  3. 點擊 "API Keys" → "Create API Key"
  4. 複製生成的 API Key
🚀 前往 Groq Console

⚙️ 2. 在此設定 API Key

🎯 3. 開始使用

設定完成後，您就可以：

  • 上傳發票圖片
  • 自動 OCR 識別
  • AI 智能格式化
  • 查看整理好的發票明細
🏠 前往 OCR 系統

💻 手動設定（開發者）

您也可以在瀏覽器控制台手動執行：

localStorage.setItem('groq_api_key', 'YOUR_API_KEY_HERE');
